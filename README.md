# PensionPilot

Swiss pension management dashboard — built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- **Dashboard** — total assets overview across all 3 pillars with growth projections
- **1st Pillar (AHV)** — contribution history, gap detection, expected pension
- **2nd Pillar (BVG)** — fund allocation, investment performance, key BVG figures
- **3rd Pillar (3a/3b)** — tax deduction tracker, account list, 3a vs 3b explainer
- **Retirement Calculator** — interactive slider-based projection with charts
- **Settings** — profile, notifications, security

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- [Next.js 14](https://nextjs.org/) — App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Recharts](https://recharts.org/) — charts
- [Lucide React](https://lucide.dev/) — icons
